Calculating the Compound Annual Growth Rate (CAGR):

\[ CAGR = \left( \frac{V_{final}}{V_{initial}} \right)^{\frac{1}{t}} - 1 \]

Where:

  • $V_{final}$ = Ending Value
  • $V_{initial}$ = Beginning Value
  • $t$ = Time in years

# Calculating Quarter Growth
def calculate_growth(current, previous):
    if previous == 0:
        return 0
    growth = ((current - previous) / previous) * 100
    return round(growth, 2)
